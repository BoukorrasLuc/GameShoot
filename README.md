# Game Shoot

- Mini shoot Game.

## About this project

- Inspired by a tutorial to find on the internet.

### Gameplay

player red:

- direction key : up arrow, down arrow, left arrow, right arrow.
- shoot : Inser or 0

player blue:

- direction key : q, d, s, z. (keyboard azerty)
- shoot : space.

- Active Ia = press "Turn On"
